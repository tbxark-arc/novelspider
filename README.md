# novel-spider
 
```
Usage of novelspider:
  -dir string
        directory to save the books (default "./")
  -help
        show help
  -interval int
        seconds between each download (default 1)
  -link string
        link of the books
  -parser string
        parser of the books, support: shuba69, sksw
  -start int
        start index of the books (default -1)
```
